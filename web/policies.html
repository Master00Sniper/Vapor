Privacy & Governance

How Vapor handles your data and how the project is maintained

Privacy Policy

Anonymous Usage Statistics

Vapor collects anonymous usage statistics to help the developer understand how many people are using the application. This data includes:

  • App start and heartbeat events
  • Vapor version number
  • Operating system type (e.g., "Windows 10")
  • A randomly generated installation ID (not linked to any personal information)

No personal information is collected. You can disable this in Settings > Preferences > "Send Anonymous Usage Statistics".

Bug Reports

Bug reports are only submitted when you explicitly click "Submit Bug Report" in the Help tab. You may optionally include:

  • System information (OS, Vapor version)
  • Recent log data (you can opt out by unchecking "Include recent logs")

Bug reports are submitted to GitHub Issues and are publicly visible. Your Windows username is automatically redacted from logs.

No Other Data Collection

Vapor does not collect, store, or transmit any other data. Your gaming activity, system configuration, and personal files are never accessed or shared.
